ROS 1 Interface Overview

Structure

The core structure of the Interbotix ROS 1 Interface is made up of three key components:

  • The ROBOTIS DYNAMIXEL Workbench Toolbox packages - Provides easy-to-use interfaces and definitions to control DYNAMIXEL servos.
  • The Interbotix X-Series Messages package - Defines messages, services, and actions to be used in other X-Series projects.
  • The Interbotix X-Series SDK package - Wraps the DYNAMIXEL Workbench Toolbox, exposing methods of control via topics and services.
