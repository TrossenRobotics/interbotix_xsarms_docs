ROS 2 InterfaceÔÉÅ

The Interbotix X-Series Arms are compatible with the following ROS 2 distributions and platforms:

  • ROS 2 Galactic on Ubuntu Linux 20.04
  • ROS 2 Humble and Rolling on Ubuntu Linux 22.04

To install the Interbotix X-Series Arm packages for any of the distributions above, go to the ROS 2 Software Setup Guide.

Once installed, follow the steps in the ROS 2 Quickstart Guide to get familiar with some of the basic features of the X-Series Arm packages.

See the ROS 2 Interface Overview for details.
